## 🧠 How It Works

- The page listens for the `keydown` event.
- When any key is pressed, it shows:
  - **Key** (actual key character),
  - **Key Code** (numeric code),
  - **Code** (physical key identifier).
- UI updates dynamically on every keystroke.
